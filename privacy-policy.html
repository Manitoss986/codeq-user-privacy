Política de Privacidad — CodeQ USER

Última actualización: 31 de octubre de 2025

1. Información general

CodeQ USER es una aplicación desarrollada por MaD_Design_Vgo que permite importar, guardar y presentar códigos QR o de barras de forma local en tu dispositivo. No requiere conexión a Internet para su funcionamiento básico ni recopila datos personales de los usuarios.

2. Datos que recopilamos

  • La aplicación no solicita ni almacena datos personales en servidores externos.
  • Los códigos que el usuario guarda permanecen únicamente en su dispositivo.
  • En caso de usar funciones opcionales como compartir por WhatsApp o correo, los datos se transmiten solo entre los dispositivos implicados.

3. Permisos

  • Almacenamiento: para importar y guardar archivos .CodeQ.
  • Cámara (opcional): solo si se activa la lectura desde cámara (no utilizada en la versión actual).
  • Galería de imágenes: para leer códigos desde imágenes seleccionadas por el usuario.

4. Servicios de terceros

CodeQ USER no usa SDKs publicitarios ni sistemas de analítica de terceros. Algunas funciones pueden usar bibliotecas locales de Google ML Kit para decodificar códigos QR o de barras. El procesamiento se realiza íntegramente en el dispositivo.

5. Conservación de datos

Todos los datos creados o importados por el usuario se almacenan localmente. El usuario puede eliminarlos en cualquier momento desde la app o desinstalando CodeQ USER.
